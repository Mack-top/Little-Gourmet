//   - game_scene.tscn: 游戏核心场景

//   - ui: UI相关纹理
//     - background.png: 主菜单背景
//     - button.png: 按钮基础纹理
//     - logo.png: 游戏logo
//     - kitchen_bg.png: 厨房场景背景
//     - ingredient_slot.png: 食材槽位图标
//     - cooking_pot.png: 烹饪锅图标
//     - chef_hat.png: 厨师帽图标
// - sounds: 存放音效文件（.wav）
// - animations: 存放动画资源（.anim）
